Crash_scoring data over different predictors is analyzed using R to illustrate a detailed [regression analysis](https://rogeliolozano.github.io/projects/linear_regression_lab.nb.html) over categorical features. Of course most of this process can be automated using Auto-ML, nevertheless, a precise knowledge of what is "under the hood" is really useful in order to confidently use tools like TPOT or Pycaret.
